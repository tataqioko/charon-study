### Added
- 🎉 首次发布
- ✨ AI 驱动的学习计划生成
- 📚 流式讲义生成（Markdown + LaTeX + Mermaid）
- 💬 智能追问系统
- 📝 6 种题型测验（单选、多选、判断、填空、简答、编程）
- 🔄 FSRS-4.5 间隔复习系统
- 📊 复习数据可视化（热力图、学习曲线、保留率）
- ❌ 智能错题本
- 📂 材料上传（支持 TXT/Markdown）
- 🎨 深色模式 + Charon 紫主题
- 🔐 Keyring 安全存储 API Key
- 📖 讲义阅读增强（字号调节、专注模式、高亮笔记、书签）
- ⚙️ 课程动态调整（压缩/扩展学习天数）
- 🧪 用户画像系统（个性化学习路径）
- 🖥️ 自定义标题栏 + 系统托盘
